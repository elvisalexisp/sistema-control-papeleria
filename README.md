# Sistema de Ventas - Internet y Papelería La Unión
Aplicación web local diseñada para la gestión integral de ventas diarias, automatización de cierres mensuales y protección de datos financieros.


Tecnologías usadas:
Backend: Python con Flask (Framework para aplicaciones web).

Frontend: HTML5 y Jinja2 (para renderizado de plantillas dinámicas).

Base de Datos y Reportes: Pandas y OpenPyxl (manejo avanzado de datos en Excel).

Automatización: Webbrowser y Threading (para apertura automática del sistema en el navegador).

Seguridad: Implementación de protección de hojas de cálculo con contraseña y lógica de encriptación de rutas para ejecutables.
